# Lorenz-Attractor
Simple 3D Visualization of Lorenz Attractor 

BUILDING (OSX):
1. To compile, in terminal, type 'make'
2. To run, type './lorenz'

KEYS SET FOR ANIMATION:
   
   ESC    		exit
   LEFT/RIGHT   change view angle (x -/+)
   UP/DOWN      change view angle (y -/+)
   s/a 		 	change view angle (z -/+)
   r 			restart animation
   f            finish animation

Note: Lorenz Attractor values by default:
	Prandtl = 10
	Rayleigh = 2.6666 or 8/3
	beta = 28
